# SPECS.md — 게임 규칙 명세

**슬레이 더 플러터 (Slay the Flutter)** — 순수 로그라이크 덱빌딩 카드 게임

---

## 1. 핵심 컨셉

플레이어는 매 런마다 기본 덱(10장)을 들고 던전에 입장한다. 전투마다 핸드에서 카드를 골라 몬스터를 처치하고, 보상으로 덱을 강화하며 점점 강해진다. 스테이지 1→2→3을 거쳐 최종 보스를 쓰러뜨리면 런 클리어. 런이 끝날 때마다 획득한 XP로 레벨이 오르고, 새로운 카드와 유물이 영구 해금되어 다음 런의 보상 풀이 넓어진다.

## 5. 몬스터

- 각 전투에는 **몬스터 1마리**만 등장한다.
- 몬스터는 매 턴 **의도(Intent)** 를 표시한다.
- 난이도는 스테이지 번호에 비례하여 선형 증가한다.

### 기본 몬스터 스탯 공식

```
몬스터 HP     = 16 + (스테이지 × 8)   → 스테이지 1: 24 / 2: 32 / 3: 40
몬스터 공격력 = 8 + (스테이지 × 2)    → 스테이지 1: 10 / 2: 12 / 3: 14
```

### 네임드 몬스터

| 이름 | 등장 스테이지 | HP | 행동 패턴 |
|------|-------------|-----|-----------|
| 끈적 슬라임 | 스테이지 1 (일반) | 35 | 분비(힘 +3) → 점착 공격(7) × 2 순환 |
| 고철수집가 | 스테이지 1 (일반) | 32 | 물기(11) → 단련(힘+2, 방어도+5) → 할퀴기(7, 방어도+3) 순환 |
| 독파수꾼 | 스테이지 2 (엘리트) | 48 | 독기 분출(4 + 취약 2턴) → 독침(7 + 취약 1턴) → 강습(12) 순환 |
| 석굴 수호자 | 스테이지 2 (엘리트) | 64 | 수면(방어도+5) × 2 → 강타(15) × 2 → 기력 흡수(8 + 약화 2턴) 순환 |
| 철갑 골렘 | 스테이지 3 (보스) | 96 | 장갑 강화(방어도+10) → 분쇄 강타(18) → 연속 타격(5×3) 순환 |

## 6. 전투 보상 시스템

- 몬스터 처치 시 즉시 보상이 지급된다.
- 보상은 **선택지 중 하나를 고르는 방식**으로 제공된다.

### 일반 몬스터 처치 보상 (3개 중 1개 선택)

| 보상 | 내용 |
|------|------|
| 카드 추가 | 해금된 카드 풀에서 무작위 3장 제시 → 1장 선택해 덱에 추가 |
| HP 회복 | 최대 HP의 25% 회복 |
| 골드 획득 | 15~30 골드 |

### 경험치(XP) 획득

| 조건 | XP |
|------|----|
| 일반 몬스터 처치 | +10 XP |
| 스테이지 클리어 | +30 XP |
| 런 클리어 (보스 처치) | +100 XP |

## 7. 전투 흐름

```
[전투 시작]
  └─ 덱 셔플 → 5장 드로우
       │
[플레이어 턴]
  └─ 에너지 3 충전
  └─ 플레이어 독 틱 (독 스택만큼 HP 감소 → 스택 1 감소)
  └─ 카드 사용 (에너지 소모)
  └─ 턴 종료 → 손패 버림, 방어도 소멸
       │
[몬스터 턴]
  └─ 의도에 따라 행동
  └─ 몬스터 독 틱 (독 스택만큼 HP 감소 → 스택 1 감소)
  └─ 몬스터 방어도 소멸, 상태 이상 duration 1 감소
       │
[반복] → 몬스터 HP 0 → 전투 승리 → 보상 획득
       → 플레이어 HP 0 → 패배 → 런 종료
```

## 8. 맵 & 노드 시스템

각 스테이지는 여러 노드로 구성된 맵으로 표현된다. 플레이어는 분기점마다 다음 노드를 선택하여 이동한다.

### 노드 종류

| 노드 | 아이콘 | 내용 |
|------|--------|------|
| 일반 몬스터 | ⚔️ | 일반 몬스터와 전투. 처치 시 보상 선택 |
| 엘리트 몬스터 | 💀 | 강화된 몬스터. 처치 시 추가 보상(유물 포함) |
| 랜덤 이벤트 | ❓ | 무작위 텍스트 이벤트. 선택지에 따라 이득 또는 손해 |
| 유물 보관소 | 🏺 | 해금된 유물 풀에서 1개 무료 획득 |
| 상점 | 🛒 | 골드로 카드 구매, 카드 제거, 유물 구매 가능 |
| 휴식처 | 🔥 | 휴식(HP 30% 회복) 또는 강화(카드 1장 업그레이드) 중 선택 |
| 챕터 보스 | 👑 | 스테이지 마지막 노드. 처치 시 다음 스테이지로 진행 |

### 맵 구조 (Act 1 — 5층 12노드 고정 레이아웃)

```
Floor 4 ──────────────────── [👑 보스]
                               ↑    ↑
Floor 3 ──────── [💀 엘리트]  [🔥 휴식]
                    ↑  ↑         ↑
Floor 2 ── [🔥 휴식] [🛒 상점] [⚔️ 몬스터]
               ↑  ↑     ↑  ↑      ↑
Floor 1 ── [⚔️] [💀 엘리트] [🏺 유물]
               ↑  ↑     ↑  ↑      ↑
Floor 0 ── [⚔️ 몬스터] [⚔️ 몬스터] [❓ 이벤트]
            (시작)       (시작)       (시작)
```

- 총 **12개 노드, 5층** 구조. 선택지는 각 노드에서 1~2갈래다.
- 연결선은 교차하지 않는다 (왼쪽 노드는 왼쪽 방향으로만 연결).
- 챕터 보스 노드(Floor 4)는 항상 마지막에 고정되어 모든 경로가 수렴한다.
- 엘리트 2개, 상점 1개, 유물 보관소 1개, 휴식처 2개 보장된다.

### 스테이지-층 매핑

| Floor | 스테이지 | 기본 몬스터 HP | 기본 공격력 |
|-------|---------|--------------|------------|
| 0, 1 | 스테이지 1 | 24 | 10 |
| 2, 3 | 스테이지 2 | 32 | 12 |
| 4 (보스) | 스테이지 3 | 40 | 14 |

### 상점 구성

| 항목 | 가격 |
|------|------|
| 카드 구매 | 50~80 골드 |
| 카드 제거 | 100 골드 |
| 유물 구매 | 100~150 골드 |
| HP 포션 | 50 골드 (최대 HP 30% 회복) |

## 9. 이벤트 시스템

이벤트 노드(❓)에서 무작위로 하나가 선택된다. 현재 등록된 이벤트는 총 **10종**이며, 각 이벤트는 **2개의 선택지**를 제공한다.

### 이벤트 목록

| 이벤트 | 선택 A | 선택 B |
|--------|--------|--------|
| 신비로운 약 | 마신다 → HP +15 | 집어서 판다 → 골드 +10 |
| 쓰러진 모험가 | 치료해준다 → 골드 +15 | 소지품을 뒤진다 → 카드 1장 |
| 악마의 계약 | 계약 수락 → 카드 1장, HP -8 | 거절 → 없음 |
| 버려진 금고 | 조심스럽게 → 골드 +20 | 세게 부순다 → 골드 +45, HP -12 |
| 고대 제단 | 헌금 챙기기 → 골드 +12 | 피를 바친다 → HP -7, 카드 1장 |
| 길 잃은 상인 | 길 안내 → 골드 +20 | 짐 빼앗기 → 카드 1장, HP -8 |
| 오래된 우물 | 물 마시기 → HP +12 | 소원 빌기 → 골드 -15, HP +25 |
| 저주받은 보물 | 손에 넣기 → 골드 +35, 카드 1장, HP -15 | 지나치기 → 없음 |
| 수상한 편지 | 뜯어 읽기 → 카드 1장 | 전달하기 → 골드 +15 |
| 무너지는 다리 | 뱃사공 부르기 → 골드 -10, HP +10 | 뛰어가기 → HP -10, 카드 1장 |

### 이벤트 규칙

- **골드 부족**: 골드를 소모하는 선택지는 보유 골드가 부족하면 비활성화된다.
- **HP 비용**: 이벤트로 인한 HP 감소는 최솟값 1로 클램프된다 (이벤트로 사망 불가).
- 선택 후 결과 화면에 서사 묘사가 표시된다.

## 10. 진행 구조 (로그라이크)

### 런(Run) 구조

- 매 런은 **기본 덱(강타 5장 + 방어 5장)** 으로 시작한다.
- 스테이지 1 → 2 → 3 → 보스 순서로 진행한다.
- 보스 처치 시 런 클리어, 패배 시 즉시 런 종료.
- 런 종료 후 획득한 XP를 합산하여 레벨업 여부를 계산한다.

### 메타 진행 (영구 성장)

- **레벨업**: XP가 임계치에 도달하면 레벨이 오른다.

| 레벨 | 필요 XP (누적) | 해금 내용 (예시) |
|------|----------------|-----------------|
| 1 | 0 | 기본 카드(강타, 방어) |
| 2 | 100 XP | 맹타(Bash), 연격(Swift Cut) |
| 3 | 250 XP | 철벽(Iron Wall), 집중(Focus) |
| 4 | 450 XP | 회복(Recover), 유물 1종 |
| 5+ | 계속 확장 | 특수 카드 및 유물 추가 해금 |

- 해금된 카드와 유물은 이후 모든 런의 보상 풀에 등장한다.
- 런 진행 중 획득한 카드/유물은 런 종료 시 초기화된다 (메타 해금 목록은 유지).
