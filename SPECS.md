# SPECS.md — 게임 규칙 명세

**슬레이 더 플러터 (Slay the Flutter)** — 순수 로그라이크 덱빌딩 카드 게임

---

## 1. 핵심 컨셉

플레이어는 매 런마다 기본 덱(10장)을 들고 던전에 입장한다. 전투마다 핸드에서 카드를 골라 몬스터를 처치하고, 보상으로 덱을 강화하며 점점 강해진다. 12층 맵을 거쳐 최종 보스를 쓰러뜨리면 런 클리어. 런이 끝날 때마다 획득한 XP로 레벨이 오르고, 새로운 카드와 유물이 영구 해금되어 다음 런의 보상 풀이 넓어진다.

## 4. 상태 이상 (Status Effects)

| 상태 | 효과 | 지속 |
|------|------|------|
| 취약 (Vulnerable) | 받는 데미지 × 1.5 (floor) | 지정 턴 수, 턴 종료 시 1 감소 |
| 약화 (Weak) | 주는 데미지 × 0.75 (floor) | 지정 턴 수, 턴 종료 시 1 감소 |
| 독 (Poison) | 스택 수만큼 HP 감소 (방어도 무시), 이후 스택 1 감소. 플레이어는 자신의 턴 시작 시, 몬스터는 플레이어 턴 종료 시 틱 발동 (§7 참조) | 스택이 0이 될 때까지 |
| 방어도 (Block) | 피해를 먼저 흡수, 턴 종료 시 소멸 | 1턴 |

## 5. 몬스터

- 각 전투에는 **몬스터 1마리**만 등장한다.
- 몬스터는 매 턴 **의도(Intent)** 를 표시한다.
- 난이도는 스테이지 번호에 비례하여 선형 증가한다.

### 기본 몬스터 스탯 공식

> 실제 게임플레이에서 등장하는 몬스터는 모두 아래 네임드 몬스터이며 고정 HP를 사용한다.
> 아래 공식은 테스트 전용 `MonsterType.basic` 타입에만 적용된다.

```
몬스터 HP     = 16 + (스테이지 × 8)   → 스테이지 1: 24 / 2: 32 / 3: 40
몬스터 공격력 = 8 + (스테이지 × 2)    → 스테이지 1: 10 / 2: 12 / 3: 14
```

### 네임드 몬스터

| 이름 | 등장 노드 | HP | 행동 패턴 |
|------|----------|----|-----------|
| 끈적 슬라임 | 일반 몬스터 | 35 | 분비(힘 +3) → 점착 공격(7) × 2 순환 |
| 고철수집가 | 일반 몬스터 | 32 | 물기(11) → 단련(힘+2, 방어도+5) → 할퀴기(7, 방어도+3) 순환 |
| 독파수꾼 | 엘리트 | 48 | 독기 분출(4 + 취약 2턴) → 독침(7 + 취약 1턴) → 강습(12) 순환 |
| 석굴 수호자 | 엘리트 | 64 | 수면(방어도+5) × 2 → 강타(15) × 2 → 기력 흡수(8 + 약화 2턴) 순환 |
| 철갑 골렘 | 보스 | 96 | 장갑 강화(방어도+10) → 분쇄 강타(18) → 연속 타격(5×3) 순환 |

> 일반 노드에서는 끈적 슬라임·고철수집가 중 무작위 등장.  
> 엘리트 노드에서는 독파수꾼·석굴 수호자 중 무작위 등장.  
> 보스 노드에서는 항상 철갑 골렘 등장.

## 6. 전투 보상 시스템

몬스터 처치 시 즉시 보상이 지급된다.

### 일반 몬스터 처치 보상

두 가지 보상이 동시에 제공된다:

1. **골드 자동 지급**: 보상 화면에서 탭하면 획득. 공식 아래 표 참고.
2. **카드 선택**: 해금된 카드 풀에서 무작위 3장 제시 → 1장 선택해 덱에 추가 (건너뛰기 가능).

### 엘리트 처치 추가 보상

일반 보상에 더해 **유물 1개** 자동 지급 (보유하지 않은 유물 중 무작위 선택).

### 보스 처치 보상

골드 없이 **유물 1개** 자동 지급 후 런 클리어.

### 골드 획득 공식

| 조건 | 공식 | 예시 (Floor 0 기준) |
|------|------|---------------------|
| 일반 몬스터 처치 | (floor + 1) + 10~14 골드 | Floor 0: 11~15 골드 |
| 엘리트 처치 | (floor + 1) + 20~25 골드 | Floor 4: 25~30 골드 |
| 보스 처치 | 0 골드 | — |

### 경험치(XP) 획득

| 조건 | XP |
|------|----|
| 일반 몬스터 처치 (승리) | +10 XP |
| 엘리트 처치 (승리) | +25 XP |
| 런 클리어 (보스 처치) | +100 XP |
| 일반 몬스터 전투 (패배) | +3 XP |
| 엘리트 전투 (패배) | +8 XP |
| 보스 전투 (패배) | +20 XP |

## 7. 전투 흐름

```
[전투 시작]
  └─ 덱 셔플
  └─ 전투 시작 유물 효과 적용 (방패 부적, 피의 유리병 등)
       │
[플레이어 턴 시작]
  └─ 에너지 3 충전 (등불·위기의 부적 등 유물 추가 적용)
  └─ 재생의 문신 유물 HP 회복 (매 턴)
  └─ 플레이어 독 틱 (독 스택만큼 HP 감소, 방어도 무시)
  └─ 카드 5장 드로우 (첫 턴: 유물 추가 드로우 포함)
       │
[플레이어 행동]
  └─ 카드 사용 (에너지 소모)
       │
[플레이어 턴 종료]
  └─ 손패 전체 버림
  └─ 턴 종료 유물 효과 (황금 방패, 냉각 보석)
  └─ 몬스터 독 틱 (독 스택만큼 HP 감소, 방어도 무시) → 스택 1 감소
  └─ 몬스터 방어도 소멸, 상태 이상 duration 1 감소
  └─ 몬스터 의도에 따라 행동
  └─ 플레이어 방어도 소멸, 상태 이상 duration 1 감소 (독 스택 포함)
       │
[반복] → 몬스터 HP 0 → 전투 승리 → 보상 획득
       → 플레이어 HP 0 → 패배 → 런 종료
```

## 8. 맵 & 노드 시스템

각 런은 **12층(Floor 0~11) 절차적 생성 맵**으로 구성된다. 플레이어는 분기점마다 다음 노드를 선택하여 이동한다.

### 노드 종류

| 노드 | 아이콘 | 내용 |
|------|--------|------|
| 일반 몬스터 | ⚔️ | 일반 몬스터와 전투. 처치 시 골드 + 카드 보상 |
| 엘리트 몬스터 | 💀 | 강화된 몬스터. 처치 시 골드 + 카드 + 유물 보상 |
| 랜덤 이벤트 | ❓ | 무작위 텍스트 이벤트. 선택지에 따라 이득 또는 손해 |
| 유물 보관소 | 🏺 | 해금된 유물 풀에서 1개 무료 획득 (건너뛰기 가능) |
| 상점 | 🛒 | 골드로 카드 구매, 카드 제거, 유물 구매 가능 |
| 휴식처 | 🔥 | 휴식(HP 30% 회복) 또는 강화(카드 1장 업그레이드) 중 선택 |
| 챕터 보스 | 👑 | 최종 노드. 처치 시 런 클리어 |

### 맵 구조 (12층 절차적 생성)

- 총 **12층(Floor 0~11)**. 층마다 1~4개의 노드가 절차적으로 생성된다.
- 모든 연결은 앞 방향(floor 증가)만 허용. 교차 없는 DAG(방향성 비순환 그래프).
- 아래 4개 층은 타입이 **고정**된다.

#### 고정 층

| Floor | 타입 | 설명 |
|-------|------|------|
| 0 | ⚔️ 일반 몬스터 | 시작 층. 모든 노드가 일반 몬스터 |
| 5 | 🏺 유물 보관소 | 항상 중간층 보관소 |
| 10 | 🔥 휴식처 | 보스 직전 강제 휴식처 |
| 11 | 👑 보스 | 최종 보스 (항상 철갑 골렘) |

#### 비고정 층 타입 확률 (Floor 1~4, 6~9)

진행도(ratio = floor / 11)에 따라 가중치가 달라진다.

| 구간 | 해당 Floor | 몬스터 | 이벤트 | 휴식처 | 상점 | 엘리트 |
|------|----------|--------|-------|-------|------|--------|
| 초반 (ratio < 0.35) | Floor 1~3 | 65% | 20% | 15% | — | — |
| 중반 (0.35 ≤ ratio < 0.70) | Floor 4, 6~7 | 28% | 15% | 20% | 20% | 17% |
| 후반 (ratio ≥ 0.70) | Floor 8~9 | 28% | — | 16% | 16% | 40% |

제약:
- Floor 0~3: 엘리트 배치 불가
- 부모·자식 노드에 엘리트·상점·휴식처 연속 배치 금지
- 엘리트 0개, 상점 0개이면 각각 강제 삽입 보장

### 스테이지-층 매핑

| Floor | 스테이지 | 기본 몬스터 HP | 기본 공격력 |
|-------|---------|--------------|------------|
| 0, 1 | 스테이지 1 | 24 | 10 |
| 2, 3 | 스테이지 2 | 32 | 12 |
| 4 이상 | 스테이지 3 | 40 | 14 |

> 네임드 몬스터(끈적 슬라임·고철수집가·독파수꾼·석굴 수호자·철갑 골렘)는 고정 HP를 사용하며 스테이지 공식을 따르지 않는다.

### 상점 구성

| 항목 | 가격 | 수량 |
|------|------|------|
| 카드 구매 | 30~45 골드 | 3장 제시 |
| 카드 제거 | 30 골드 | 1회 한정 |
| 유물 구매 | 80~110 골드 | 2개 제시 |

## 9. 이벤트 시스템

이벤트 노드(❓)에서 무작위로 하나가 선택된다. 현재 등록된 이벤트는 총 **10종**이며, 각 이벤트는 **2개의 선택지**를 제공한다.

### 이벤트 목록

| 이벤트 | 선택 A | 선택 B |
|--------|--------|--------|
| 신비로운 약 | 마신다 → HP +15 | 집어서 판다 → 골드 +10 |
| 쓰러진 모험가 | 치료해준다 → 골드 +15 | 소지품을 뒤진다 → 카드 1장 |
| 악마의 계약 | 계약 수락 → 카드 1장, HP -8 | 거절 → 없음 |
| 버려진 금고 | 조심스럽게 → 골드 +20 | 세게 부순다 → 골드 +45, HP -12 |
| 고대 제단 | 헌금 챙기기 → 골드 +12 | 피를 바친다 → HP -7, 카드 1장 |
| 길 잃은 상인 | 길 안내 → 골드 +20 | 짐 빼앗기 → 카드 1장, HP -8 |
| 오래된 우물 | 물 마시기 → HP +12 | 소원 빌기 → 골드 -15, HP +25 |
| 저주받은 보물 | 손에 넣기 → 골드 +35, 카드 1장, HP -15 | 지나치기 → 없음 |
| 수상한 편지 | 뜯어 읽기 → 카드 1장 | 전달하기 → 골드 +15 |
| 무너지는 다리 | 뱃사공 부르기 → 골드 -10, HP +10 | 뛰어가기 → HP -10, 카드 1장 |

### 이벤트 규칙

- **골드 부족**: 골드를 소모하는 선택지는 보유 골드가 부족하면 비활성화된다.
- **HP 비용**: 이벤트로 인한 HP 감소는 최솟값 1로 클램프된다 (이벤트로 사망 불가).
- 선택 후 결과 화면에 서사 묘사가 표시된다.

## 10. 진행 구조 (로그라이크)

### 런(Run) 구조

- 매 런은 **기본 덱(강타 5장 + 방어 5장)** 과 **75골드** 로 시작한다.
- 12층 맵을 진행하며 보스를 처치하면 런 클리어, 패배 시 즉시 런 종료.
- 런 종료 후 획득한 XP를 합산하여 레벨업 여부를 계산한다.

### 메타 진행 (영구 성장)

- **레벨업**: XP가 임계치에 도달하면 레벨이 오른다.

| 레벨 | 필요 누적 XP | 신규 해금 카드 |
|------|--------------|----------------|
| 1 | 0 | 강타(Strike), 방어(Defend) |
| 2 | 100 XP | 세 번 베기(Triple Slash), 독침(Toxic Jab), 연속 강타(Combo Strike) |
| 3 | 250 XP | 맹타(Bash), 철벽(Iron Wall) |
| 4 | 450 XP | 집중(Focus), 회복(Recover), 불굴(Indomitable) |
| 5 | 700 XP | 취약 틈새(Exploit Weakness), 약화 강타(Weak Slash), 방어도 공격(Block Strike) |
| 6 | 1000 XP | 독화살(Poison Dart), 전투 함성(Battle Cry) |
| 7 | 1350 XP | 파괴의 일격(Crushing Blow), 혈기(Blood Rush) |
| 8 | 1750 XP | 악마의 거래(Devil's Deal), 승부수(Gamble) |
| 9 | 2200 XP | 극한 각성(Limit Break), 업화(Fiend Fire) |
| 10 | 2700 XP | 폭격 태세(Double Tap), 철옹성(Impervious) |

- 연격(Swift Cut), 광분(Rage Burst), 응급처치(Quick Mend), 결집(Regroup), 날렵한 방어(Swift Guard),
  무기 연마(Sharpen), 분노(Fury) 7종은 레벨과 무관하게 처음부터 보상 풀에 포함된다.
- 해금된 카드는 이후 모든 런의 보상 풀에 등장한다.
- 유물은 레벨과 무관하게 처음부터 20종 전체가 보상 풀에 포함된다 (11장 참고).
- 런 진행 중 획득한 카드/유물은 런 종료 시 초기화된다 (메타 해금 목록은 유지).

## 11. 유물 (Relics)

**레벨과 무관하게 20종 전체가 항상 보상 풀에 포함된다**. 보유 유물은 런 종료 시 초기화된다.
