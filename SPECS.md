# SPECS.md — 게임 규칙 명세

**슬레이 더 플러터 (Slay the Flutter)** — 덱빌딩 로그라이크 일과 관리 앱

---

## 1. 핵심 컨셉

현실의 할 일(Todo)을 카드 전투로 처리한다. 사용자는 매일 '던전'에 입장하여 카드를 사용해 몬스터를 처치하고, 퀘스트(실제 할 일)를 완료함으로써 보상을 얻는다.

---

## 2. 캐릭터 (고정 1인)

| 속성 | 기본값 | 설명 |
|------|--------|------|
| 최대 체력 | 70 HP | 세션 간 유지됨 |
| 초기 덱 크기 | 10장 | 공격 5장 + 방어 5장 |
| 턴당 에너지 | 3 | 카드 사용 비용 |
| 턴당 드로우 | 5장 | 매 전투 턴 시작 시 |

---

## 3. 카드 종류 (미니멀 기본 세트)

### 공격 카드

| 이름 | 비용 | 효과 | 설명 |
|------|------|------|------|
| 강타 (Strike) | 1 | 6 데미지 | 기본 공격 |
| 맹타 (Bash) | 2 | 8 데미지 + 취약(2턴) | 강력한 일격 |
| 연격 (Swift Cut) | 1 | 4 데미지 × 2회 | 빠른 두 번 공격 |

### 방어 카드

| 이름 | 비용 | 효과 | 설명 |
|------|------|------|------|
| 방어 (Defend) | 1 | 방어도 5 획득 | 기본 방어 |
| 철벽 (Iron Wall) | 2 | 방어도 10 획득 | 강력한 방어 |

### 특수 카드 (보상으로만 획득)

| 이름 | 비용 | 효과 | 설명 |
|------|------|------|------|
| 집중 (Focus) | 0 | 다음 카드 효과 +50% | 버프 |
| 회복 (Recover) | 2 | HP 8 회복 | 전투 중 힐 |

## 4. 상태 이상 (Status Effects)

| 상태 | 효과 | 지속 |
|------|------|------|
| 취약 (Vulnerable) | 받는 데미지 +50% | 지정 턴 수 |
| 약화 (Weak) | 주는 데미지 -25% | 지정 턴 수 |
| 방어도 (Block) | 해당 수치만큼 피해 흡수, 턴 종료 시 소멸 | 1턴 |

## 5. 몬스터 (단순화)

- 각 전투에는 **몬스터 1마리**만 등장한다.
- 몬스터는 매 턴 **의도(Intent)** 를 표시한다: 공격 or 방어 중 하나.
- 난이도는 스테이지 번호에 비례하여 선형 증가한다.

### 몬스터 스탯 공식

```
몬스터 HP = 20 + (스테이지 × 10)
몬스터 공격력 = 8 + (스테이지 × 2)
```

## 6. 전투 보상 시스템

- 몬스터 처치 시 즉시 보상이 지급된다.
- 스테이지 클리어(보스 처치) 시 추가 보상이 지급된다.
- 보상은 **선택지 중 하나를 고르는 방식**으로 제공된다.

### 일반 몬스터 처치 보상 (3개 중 1개 선택)

| 보상 | 내용 |
|------|------|
| 카드 추가 | 해금된 카드 풀에서 무작위 3장 제시 → 1장 선택해 덱에 추가 |
| HP 회복 | 최대 HP의 25% 회복 |
| 골드 획득 | 15~30 골드 (이후 상점에서 사용) |

### 스테이지 클리어 보상 (추가 지급)

| 보상 | 내용 |
|------|------|
| 유물 선택 | 해금된 유물 풀에서 무작위 2개 제시 → 1개 선택 |
| 카드 제거 | 덱에서 카드 1장 영구 제거 (덱 압축) |

### 경험치(XP) 획득

| 조건 | XP |
|------|----|
| 일반 몬스터 처치 | +10 XP |
| 스테이지 클리어 | +30 XP |
| 런 클리어 (보스 처치) | +100 XP |

---

## 7. 전투 흐름

```
[전투 시작]
  └─ 덱 셔플 → 5장 드로우
       │
[플레이어 턴]
  └─ 에너지 3 충전
  └─ 카드 사용 (에너지 소모)
  └─ 턴 종료 → 손패 버림, 방어도 소멸
       │
[몬스터 턴]
  └─ 의도에 따라 행동 (공격 or 방어)
       │
[반복] → 몬스터 HP 0 → 전투 승리 → 보상 획득
                    → 플레이어 HP 0 → 패배 → 세션 종료
```

## 8. 맵 & 노드 시스템

각 스테이지는 여러 노드로 구성된 맵으로 표현된다. 플레이어는 분기점마다 다음 노드를 선택하여 이동한다.

### 노드 종류

| 노드 | 아이콘 | 내용 |
|------|--------|------|
| 일반 몬스터 | ⚔️ | 일반 몬스터와 전투. 처치 시 보상 선택 |
| 엘리트 몬스터 | 💀 | 강화된 몬스터. 처치 시 추가 보상(유물 포함) |
| 랜덤 이벤트 | ❓ | 무작위 텍스트 이벤트. 선택지에 따라 이득 또는 손해 |
| 유물 보관소 | 🏺 | 해금된 유물 풀에서 1개 무료 획득 |
| 상점 | 🛒 | 골드로 카드 구매, 카드 제거, 유물 구매 가능 |
| 휴식처 | 🔥 | 휴식(HP 회복) 또는 강화(카드 1장 업그레이드) 중 선택 |
| 챕터 보스 | 👑 | 스테이지 마지막 노드. 처치 시 다음 스테이지로 진행 |

### 맵 구조 예시 (스테이지 1)

```
[시작]
  ├─ ⚔️ → ❓ → 🔥
  ├─ ⚔️ → ⚔️ → 🛒
  └─ ❓ → 💀 → 🏺
              └──────→ [👑 챕터 보스]
```

- 각 스테이지는 **6~8개 노드**로 구성되며 분기는 2~3갈래다.
- 챕터 보스 노드는 항상 스테이지 마지막에 고정된다.
- 엘리트, 상점, 유물 보관소는 스테이지당 **최소 1개** 보장된다.

### 상점 구성

| 항목 | 가격 |
|------|------|
| 카드 구매 | 50~80 골드 |
| 카드 제거 | 100 골드 |
| 유물 구매 | 100~150 골드 |
| HP 포션 | 50 골드 (최대 HP 30% 회복) |

## 9. 진행 구조 (로그라이크)

### 런(Run) 구조

- 매 런은 **기본 덱(강타 5장 + 방어 5장)** 으로 시작한다.
- 스테이지 1 → 2 → 3 → 보스 순서로 진행한다.
- 보스 처치 시 런 클리어, 패배 시 즉시 런 종료.
- 런 종료 후 획득한 XP를 합산하여 레벨업 여부를 계산한다.

### 메타 진행 (영구 성장)

- **레벨업**: XP가 임계치에 도달하면 레벨이 오른다.

| 레벨 | 필요 XP (누적) | 해금 내용 (예시) |
|------|----------------|-----------------|
| 1 | 0 | 기본 카드(강타, 방어) |
| 2 | 100 XP | 맹타(Bash), 연격(Swift Cut) |
| 3 | 250 XP | 철벽(Iron Wall), 집중(Focus) |
| 4 | 450 XP | 회복(Recover), 유물 1종 |
| 5+ | 계속 확장 | 특수 카드 및 유물 추가 해금 |

- 해금된 카드와 유물은 이후 모든 런의 보상 풀에 등장한다.
- 런 진행 중 획득한 카드/유물은 런 종료 시 초기화된다 (메타 해금 목록은 유지).
